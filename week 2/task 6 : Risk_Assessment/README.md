# Risk Assessment Practice (ALE & Risk Matrix)

## Project Overview
This project demonstrates basic risk assessment techniques using **Annualized Loss Expectancy (ALE)** and a **5x5 Risk Matrix**.  
The goal is to evaluate the financial and operational risk of a **ransomware scenario**.

---

## Tools Used
- Google Sheets  
- Basic Risk Assessment Methodology  

---

## Steps Performed
1. Created a spreadsheet in Google Sheets  
2. Entered values for SLE and ARO  
3. Calculated ALE using formula  
4. Designed a 5x5 Risk Matrix (Likelihood vs Impact)  
5. Plotted ransomware scenario in the matrix  
6. Applied color coding for risk levels  

---

## Formula Used
ALE = SLE × ARO
SLE = $10,000
ARO = 0.2
ALE = $2,000


---

## Risk Matrix Details

| Parameter   | Value        |
|------------|--------------|
| Likelihood | 2 (Low)      |
| Impact     | 4 (High)     |
| Risk Score | 8            |
| Risk Level | Medium (Yellow) |

---

## Output
- The ransomware scenario is categorized as a **medium-level risk**  
- Expected annual loss (**ALE**) = **$2,000**

## Conclusion
This project shows how organizations can:
- Estimate potential losses

Demonstrates practical **risk assessment using simple tools**
