# 🔐 Password Security

## Overview

This task focused on understanding password security by generating strong passwords and testing weak password vulnerabilities using automated tools.

---

## What was done

A secure password vault was created, and strong passwords were generated using a password manager. Additionally, a weak password test was performed to evaluate how easily simple credentials can be compromised.

---

## Why it was done

The purpose of this activity was to understand the importance of strong passwords and how weak credentials can be exploited by attackers using automated tools.

---

## Importance

Password security is a critical aspect of cybersecurity. Weak passwords can be easily cracked, leading to unauthorized access. Strong password practices and secure storage significantly reduce the risk of compromise.

---

## Insight

Automated tools can quickly test weak passwords, demonstrating that simple or common credentials are highly vulnerable. Strong, complex passwords are essential for system security.

---

## Conclusion

The task highlighted the importance of using strong passwords and avoiding weak credentials to protect systems from unauthorized access.
