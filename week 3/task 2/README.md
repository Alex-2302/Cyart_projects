# Phishing Attack Simulation & Detection  

---

## Objective
The objective of this task is to understand phishing attacks by simulating a controlled phishing campaign using GoPhish and analyzing detection and prevention techniques.

---

## Tools Used
- Kali Linux  
- GoPhish  
- Mailtrap (SMTP Testing)  
- VirtualBox  

---

## Task Description
In this task, a phishing attack simulation was performed using GoPhish. A phishing email template and a fake login landing page were created, and a campaign was launched to track user interactions such as email opens, link clicks, and credential submissions.

---

## Working Steps

1. Studied phishing attack concepts and social engineering techniques.  
2. Configured GoPhish on Kali Linux.  
3. Created a phishing email template.  
4. Designed a fake login landing page for credential capture.  
5. Configured SMTP using Mailtrap for safe email delivery.  
6. Created a user group with test email accounts.  
7. Launched a phishing campaign using GoPhish.  
8. Monitored campaign results including:
   - Email delivery  
   - Link clicks  
   - Credential submissions  
9. Analyzed phishing indicators and user behavior.  
10. Documented prevention techniques and awareness outcomes.  

---

## Results
- Phishing emails were successfully simulated.  
- User interaction (opens, clicks, submissions) was tracked.  
- The campaign dashboard provided detailed analytics.  
- The simulation demonstrated how users can be targeted through social engineering.  

---

## Phishing Red Flags Identified
- Urgent or threatening language  
- Generic greetings (e.g., "Dear User")  
- Suspicious or unexpected links  
- Requests for sensitive information  
- Unverified sender addresses  

---

## Prevention Methods
- Security awareness training  
- Email filtering and spam detection  
- Verifying sender email addresses  
- Hovering over links before clicking  
- Using Multi-Factor Authentication (MFA)  
- Reporting suspicious emails
